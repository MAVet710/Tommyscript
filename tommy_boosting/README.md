# Tommy Boosting
Tommy Boosting is a full FiveM vehicle boosting resource with contracts, progression, crypto store, VIN scratch, leaderboard, history, and admin management.
## Features
- Contract classes D to S+
- oxmysql + ox_lib callbacks
- QBCore/ESX/standalone bridge
- NUI dashboard/contracts/store/history/leaderboard
- Anti-exploit validation
## Dependencies
oxmysql, ox_lib. Optional: qb-core, es_extended, ox_inventory, ox_target/qb-target, dispatch scripts.
## Commands
- /boosting
## Items
boosting_laptop, tracker_remover, hacking_device, vin_scratcher, advanced_lockpick
## Exports/Events
Implemented with `tommy_boosting:*` namespaced events and exports in server/client main.
